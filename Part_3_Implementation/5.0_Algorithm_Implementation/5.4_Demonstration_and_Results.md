# 5.4 Demonstration

The program is demonstrated using three scenarios that reflect the real-world landslide emergency on 14 November 2025 in Hulu Selangor.

## Input: Post-Landslide Road Network

The road network is represented as a weighted graph with 8 nodes and 9 surviving edges.

| Edge | Description | Weight (min) |
|---|---|---|
| 0 → 1 | Hospital KKB → Pekan KKB Junction | 8 |
| 1 → 2 | Pekan KKB → Batang Kali Fork | 10 |
| 1 → 3 | Pekan KKB → Sungai Chilling Entrance | 18 |
| 2 → 4 | Batang Kali → Ladang Ulu Bernam | 20 |
| 2 → 5 | Batang Kali → Pos Gertak (damaged) | 36 |
| 3 → 5 | Sungai Chilling → Pos Gertak (forest road) | 17 |
| 4 → 6 | Plantation → Hulu Bernam Crossroad | 10 |
| 5 → 7 | Pos Gertak → Kg. Sungai Lui | 12 |
| 6 → 7 | Hulu Bernam → Kg. Sungai Lui | 14 |

> **Note:** The direct main highway (Node 0 → Node 7) is completely blocked by the landslide and is omitted from the graph.

---

**Expected Output:**

```

╠══════════════════════════════════════════════════════════════════╣

║  Destination : Node 7  — Kampung Sungai Lui

║  Segment Breakdown
║  From → To   Time (min)   Road Segment
║  ──────────────────────────────────────────────────────────────
║    0 → 1       8 min     Hospital KKB → Pekan KKB Junction
║    1 → 3      18 min     Pekan KKB → Sungai Chilling
```

**Interpretation:** Dijkstra's Algorithm correctly identifies the path via Sungai Chilling Forest Reserve (total: 55 min) as faster than the plantation route via nodes 2→4→6→7 (total: 62 min).

---

## Scenario B: Additional Road Blockage (Edge 1→3 Removed)

If the Sungai Chilling road is also found blocked upon arrival at Node 1:

**Expected Output:**
```
Shortest Travel Time : 62 minutes
Optimal Route        : 0 → 1 → 2 → 4 → 6 → 7
```

Dijkstra's automatically finds the next best route — through the plantation estate — without any manual adjustment.

---

## Scenario C: Village Completely Isolated

If all connecting roads are blocked and no path from hospital to village exists:

**Expected Output:**
```
✘ NO PATH FOUND — village is completely cut off.
→ Recommend: RMAF/AADK helicopter evacuation.
```

# 5.5 Results

## All Shortest Distances from Hospital (Scenario A)

| Node | Location | Shortest Time |
|---|---|---|
| 0 | Hospital Kuala Kubu Bharu | 0 min (source) |
| 1 | Pekan KKB Junction | 8 min |
| 2 | Batang Kali Fork | 18 min |
| 3 | Sungai Chilling Entrance | 26 min |
| 4 | Ladang Ulu Bernam | 38 min |
| 5 | Pos Gertak | 43 min |
| 6 | Hulu Bernam Crossroad | 48 min |
| **7** | **Kg. Sungai Lui** | **55 min** |

## Key Findings

1. **Optimal Path:** Hospital → Pekan KKB → Sungai Chilling → Pos Gertak → Kg. Sungai Lui (4 stops, 55 minutes total).
2. **Suboptimal Alternative:** Hospital → Pekan KKB → Batang Kali → Plantation → Hulu Bernam → Village (5 stops, 62 minutes) — 7 minutes slower.
3. **Landslide Impact:** Without the direct highway (normally ~20 min direct), the best detour adds ~35 minutes of extra travel time.
4. **Medical Significance:** 55 minutes is within the critical 90-minute cardiac emergency window, making evacuation viable if the optimal route is followed. A suboptimal route might not leave enough time for intervention.
5. **Adaptability:** Removing additional blocked edges (Scenario B) automatically yields the next best route — demonstrating Dijkstra's robustness for dynamic disaster conditions.
